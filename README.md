# Investment Portfolio Dashboard

Dual-purpose investment portfolio dashboard for SGX Dividend Income & US Momentum Trading.

- Real-time market data from Yahoo Finance
- Technical analysis: EMA, SMA, RSI, VWAP, candlestick patterns
- Alpha Signal logic with entry/exit/stop-loss recommendations
- Portfolio P&L tracking with dividend income projections
